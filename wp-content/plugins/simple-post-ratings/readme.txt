== Description ==

A simple plugin for adding ratings to your blog posts. An easy and intuitive interface lets you modify a number of options including
 rating scale, position, custom rating images etc.


== Installation ==

1. Upload the simple-post-ratings.zip using WordPress plugins uploader.
2. Activate the plugin
3. Under Settings/Simple Post Ratings Options modify your rating options
4. Create a new post or edit an existing one
5. Find Post Rating box on the right hand side and add rating to your post
6. Publish
7. Enjoy!
